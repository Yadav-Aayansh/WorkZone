## 1. Project Overview

WorkZone.tech is a comprehensive, Generative-AI–powered HR Management Platform designed for startups and mid-sized companies. It enables organizations to automate recruitment, manage employee data, streamline communication, generate intelligent insights, and improve overall HR operations through modern, scalable technology.

This platform is developed as part of the Software Engineering Project – Team 10.

### 1.1. Key Features

- Company-specific subdomain (e.g., `company.workzone.tech`)
- Personalise workspace (e.g., `hrm.<company's official domain>`)
- AI Resume Scoring & Feedback
- AI Interview Assistant
- Automated Document Generator
- AI Helpdesk & Ticketing System
- Employee & Manager Dashboards
- Recruiter Portal with automated workflows
- Applicant Job Portal
